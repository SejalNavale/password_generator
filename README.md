## 2. Password Generator

### Description
This program generates a secure random password based on user-defined
preferences such as length and character types.

### Features
- User-defined password length
- Option to include uppercase letters, lowercase letters, numbers, and special characters
- Random password generation
- Secure storage of passwords using SHA-256 encryption

### Concepts Used
- Random module
- String manipulation
- Hashing using hashlib
- User input validation
